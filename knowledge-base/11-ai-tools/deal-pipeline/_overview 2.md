# Deal Pipeline Tools

## Overview

Tools for managing deals through your pipeline from prospect to close.

**Total Tools**: 12

## Available Tools


### createDeal

**ID**: `deal.create`

Create a new deal in the pipeline


### updateDealStage

**ID**: `deal.update_stage`

Move a deal to a different pipeline stage


### analyzeDealProgress

**ID**: `deal.analyze_progress`

Analyze deal progress and identify bottlenecks


### generateOfferStrategy

**ID**: `deal.generate_offer_strategy`

Calculate tiered offer prices (optimal, target, max, walk-away)


### assignBuyerToDeal

**ID**: `deal.assign_buyer`

Assign a buyer to a deal and move to assigned stage


### getDealTimeline

**ID**: `deal.get_timeline`

Get activity timeline for a deal


### predictDealOutcome

**ID**: `deal.predict_outcome`

Predict likelihood of deal closing successfully


### generateDealSummary

**ID**: `deal.generate_summary`

Generate a comprehensive summary of a deal


### compareDealToPortfolio

**ID**: `deal.compare_to_portfolio`

Compare deal metrics to portfolio averages


### suggestDealActions

**ID**: `deal.suggest_actions`

Suggest next actions for a deal


### calculateDealMetrics

**ID**: `deal.calculate_metrics`

Calculate financial metrics for a deal


### flagDealIssues

**ID**: `deal.flag_issues`

Identify deals with issues that need attention


## Common Use Cases

- Track deal stages
- Manage deal activities
- Forecast pipeline
