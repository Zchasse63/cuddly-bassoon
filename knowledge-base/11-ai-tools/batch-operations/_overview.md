# Batch Operations Tools

## Overview

Tools for processing multiple records at once.

**Total Tools**: 4

## Available Tools


### Bulk Skip Trace

**ID**: `batch.skip_trace_bulk`

Queue multiple properties for skip tracing. Returns a job ID to track progress.


### Bulk Add to List

**ID**: `batch.add_to_list_bulk`

Add multiple properties to a list at once with optional tags.


### Bulk Update Deal Status

**ID**: `batch.update_deal_status`

Update the status of multiple deals at once.


### Export Properties

**ID**: `batch.export_properties`

Export properties to CSV, Excel, or JSON format.


## Common Use Cases

- Batch skip trace
- Bulk updates
- Mass operations
