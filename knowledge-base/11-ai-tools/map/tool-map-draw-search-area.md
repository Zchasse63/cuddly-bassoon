# Draw Search Area

## Quick Reference

| Property | Value |
|----------|-------|
| **Tool ID** | `map.draw_search_area` |
| **Category** | Map & Geographic Tools |
| **Permission** | read |
| **Confirmation Required** | No |
| **Typical Duration** | 2000ms |
| **Rate Limit** | 30 calls/minute |

## Description

Create a search area on the map by defining a center point and radius, or drawing a polygon. Returns the bounds and property count within the area.

## When to Use

Use **Draw Search Area** when you need to:
- Create heat maps
- Analyze areas
- Find specific records matching criteria
- Create new records or content
