## Description

Query points of interest (POIs) near a location using Mapbox Tilequery API. Find schools, restaurants, shops, and other amenities.

## When to Use

Use **Spatial Query** when you need to:
- Create heat maps
- Analyze areas
- Find specific records matching criteria
