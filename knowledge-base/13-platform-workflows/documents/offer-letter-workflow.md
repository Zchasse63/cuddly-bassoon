# Offer Letter Workflow

## Overview

Generate professional purchase offer letters quickly using property data and your calculated offer amounts. This workflow covers creating, customizing, and sending offer letters.

## Generating Basic Offer Letters

### Quick Offer Letter
```
"Generate an offer letter for [address] at $[amount]"
```

### Include Analysis Data
```
"Create an offer letter for [address]:
- Offer amount: $145,000
- Include property details
- Add justification based on my analysis"
```

## Offer Letter Components

### Standard Elements
| Element | Description |
|---------|-------------|
| Property Address | Subject property details |
| Offer Amount | Your proposed purchase price |
| Earnest Money | Deposit amount |
| Contingencies | Inspection, financing, etc. |
| Closing Timeline | Proposed close date |
| Expiration | Offer validity period |

### Request Specific Format
```
"Generate offer letter with:
- Cash offer, no financing contingency
- 10-day inspection period
- 21-day closing
- $2,500 earnest money
- 7-day offer expiration"
```

## Offer Letter Types

### Standard Purchase Offer
```
"Create standard purchase offer for [address]"
```
- Full contingencies
- Standard closing timeline
- Bank financing option

### Cash Offer Letter
```
"Create cash offer letter for [address]"
```
- No financing contingency
- Quick closing (14-21 days)
- Proof of funds mention

### As-Is Offer
```
"Create as-is offer letter for [address]"
```
- No repair requests
- Shortened inspection (informational only)
- Discount reflected in price

### Subject-To Offer
```
"Create subject-to existing financing offer"
```
- Taking over existing mortgage
- Special contingencies
- Different structure

## Offer Letter Best Practices

### Professionalism
- Use clean, professional format
- Correct spelling and grammar
- Accurate property details
- Clear terms and conditions

### Clarity
- Unambiguous offer amount
- Specific contingencies and deadlines
- Clear next steps for seller

### Compliance
- Follow local contract laws
- Include required disclosures
- Use appropriate legal language
- Consider attorney review for complex deals
